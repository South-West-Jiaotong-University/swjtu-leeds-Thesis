> 这是用于利兹学院论文的 $\LaTeX$ 模板

# 原版简介～
修改了Jon Shiach的模板以适应利兹大学推荐的论文格式。所有的功劳都归于Jon Shiach。

模板显示的是计算机科学，但该格式建议用于所有课程。
